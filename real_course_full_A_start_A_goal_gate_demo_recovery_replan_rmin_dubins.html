考え方

通常時: original_gates を順番に追従 逸脱時: 1. 現在位置を元パスへ射影し横ずれを判定 2. 残りゲートから復帰しやすいゲートを探索 3. 現在位置・方位 → 復帰先ゲート姿勢をRmin制約つきDubins経路で接続 4. Dubins経路上に復帰ゲートを配置 5. recovery_gates + original_gates[復帰先以降] を一時パスにする 復帰ゲートを抜けたら: 元のゲート列追従へ戻る
